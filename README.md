# Handoo Landing Static

Archivos incluidos:

- `index.html`
- `styles.css`

No necesita backend, Supabase ni build.

## Para probar en local

Abre `index.html` directamente en el navegador.

## Para subir a GitHub

1. Crea un repositorio, por ejemplo `handoo-landing`.
2. Sube `index.html` y `styles.css` a la raíz del repo.
3. En Render crea un Static Site conectado a ese repo.
4. Build command: dejar vacío.
5. Publish directory: `.`

## Dominio

Cuando Render genere la URL `*.onrender.com`, añade los dominios:
- `handoo.app`
- `www.handoo.app`

Después configura DNS según te indique Render.
